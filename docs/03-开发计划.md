## 文件结构

```
entry/src/main/ets/
├── entryability/
│   └── EntryAbility.ets                 # ✅ 应用入口
├── pages/
│   ├── Index.ets                        # ✅ 主页（难度选择 + 种子输入 + 对称选择）
│   ├── GamePage.ets                     # ✅ 游戏页
│   ├── StatsPage.ets                    # ✅ 统计/历史页面（两 Tab）
│   ├── ImportPreviewPage.ets            # [P3] OCR 预览页
│   └── MultiplayerLobbyPage.ets         # [P4] 多人大厅页
├── components/
│   ├── SudokuBoard.ets                  # ✅ 棋盘组件（9×9 网格）
│   ├── SudokuCell.ets                   # ✅ 单元格组件（数字/笔记渲染 + 填数动画）
│   ├── GestureNumberPad.ets             # ✅ 3×3 手势键盘
│   ├── GestureKey.ets                   # ✅ 单个手势键（PanGesture）
│   ├── GestureModeIndicator.ets         # ✅ 手势模式提示条
│   ├── AuxButtonGrid.ets               # ✅ 辅助按钮区（撤销/擦除/提示等）
│   ├── TopBar.ets                       # ✅ 顶部信息栏（难度/计时/错误数）
│   ├── DifficultySegment.ets            # ✅ 难度分段按钮（含技巧描述动画）
│   ├── SettingsPanel.ets                # ✅ 设置面板（左右手切换）
│   ├── GestureGuideOverlay.ets          # ✅ 首次使用手势引导遮罩
│   └── CoachSheet.ets                   # ✅ AI 教练对话 Sheet
├── models/
│   ├── CellData.ets                     # ✅ Cell 接口、CellFlags、NoteState、操作函数
│   ├── GameState.ets                    # ✅ 游戏状态模型（棋盘、答案、计时、错误数、种子）
│   └── GameHistory.ets                  # ✅ 操作历史栈（撤销/重做）
├── utils/
│   ├── SudokuGenerator.ets              # ✅ 生成器（终盘 + 对称挖洞 + 难度匹配重试 + Seed）
│   ├── SudokuSolver.ets                 # ✅ 暴力求解器（Norvig 风格 CP+DFS）
│   ├── LogicSolver.ets                  # ✅ 逻辑求解器（6 种技巧识别，提示用）
│   ├── SudokuValidator.ets              # ✅ 验证工具（冲突检测、完成判定）
│   ├── DifficultyEvaluator.ets          # ✅ 难度评估（技巧加权评分）
│   ├── Topology.ets                     # ✅ 预计算 peers/units/boxes
│   ├── Common.ets                       # ✅ 通用工具（shuffle、cloneBoard、SeededRandom）
│   ├── GameStorage.ets                  # ✅ 游戏存档持久化（Preferences）
│   ├── StatsStorage.ets                 # ✅ 统计数据持久化
│   ├── HistoryStorage.ets               # ✅ 历史记录持久化
│   ├── SettingsStorage.ets              # ✅ 设置持久化（左右手偏好）
│   └── HandDetector.ets                 # ✅ 左右手检测（motion.HoldingHandStatus）
├── constants/
│   └── GameConstants.ets                # ✅ 常量（Colors、Sizes、Difficulty、DifficultyConfig）
└── services/
    ├── CoachService.ets                 # ✅ AI 教练（localChatModel 接入）
    ├── OCRService.ets                   # [P3] 拍照识别
    ├── DistributedSyncService.ets       # [P4] 跨设备同步
    └── MultiplayerService.ets           # [P4] 多人游戏
```

## 开发顺序建议

```
第 1 步：数据模型 + 常量定义（1.1, 1.2, 1.8）
    ↓ 基础搭好
第 2 步：核心算法（1.3, 1.4, 1.5, 1.6, 1.7）
    ↓ 用 ohosTest 单元测试验证正确性
第 3 步：单元格 + 棋盘 UI（1.9, 1.10）
    ↓ 先用写死数据测试渲染
第 4 步：手势键盘（1.11, 1.12, 1.13）
    ↓ 核心交互
第 5 步：辅助区 + 页面组装（1.14, 1.15, 1.16, 1.17, 1.18）
    ↓
第 6 步：连接算法和 UI（1.19, 1.20）
    ↓ MVP 完成 🎉
第 7 步：逐步推进 Phase 2 → 3 → 4
```
